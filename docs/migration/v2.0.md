# Migration Guide: V2.0

## Overview

VQVAETask and MaskGITTask have been updated to support an **injected component pattern**. Legacy scalar/config constructor arguments are now deprecated in favor of pre-assembled components.

## Breaking Changes

### VQVAETask Constructor

**Before (deprecated):**

```python
task = VQVAETask(
    model_config=model_cfg,
    optimizer_config=optimizer_cfg,
    in_channels=1,
    out_channels=1,
    latent_channels=256,
    num_embeddings=8192,
    embedding_dim=256,
    num_channels=(64, 128, 256),
    num_res_blocks=(2, 2, 2),
    attention_levels=(False, False, False),
    lr_g=4.5e-06,
    lr_d=1e-04,
    lambda_l1=1.0,
    lambda_vq=1.0,
    lambda_gan=0.1,
    use_perceptual=True,
    lambda_perceptual=0.1,
    perceptual_network="alex",
    disc_start=0,
    disc_factor=1.0,
    use_adaptive_weight=True,
    adaptive_weight_max=100.0,
    disc_loss="hinge",
    sliding_window={"enabled": True, "roi_size": [128, 128, 128]},
    commitment_cost=0.25,
    gradient_clip_val=1.0,
    gradient_clip_enabled=True,
    quantizer_type="vq",
    fsq_levels=(8, 8, 8, 5, 5, 5),
    num_splits=None,
    dim_split=1,
    data_config=data_cfg,
)
```

**After (injected pattern):**

```python
from maskgit3d.runtime.composition import build_vqvae_task
from omegaconf import OmegaConf

cfg = OmegaConf.create({
    "task": {
        "_target_": "maskgit3d.tasks.vqvae_task.VQVAETask",
        "use_perceptual": False,
        "gradient_clip_enabled": True,
        "gradient_clip_val": 1.0,
    },
    "model": {
        "_target_": "maskgit3d.models.vqvae.VQVAE",
        "in_channels": 1,
        "out_channels": 1,
        "latent_channels": 256,
        "num_embeddings": 8192,
        "embedding_dim": 256,
        "num_channels": [64, 128, 256],
        "num_res_blocks": [2, 2, 2],
        "attention_levels": [False, False, False],
        "commitment_cost": 0.25,
        "quantizer_type": "vq",
    },
    "optimizer": {"_target_": "torch.optim.Adam", "lr": 1e-4},
    "data": {"crop_size": [128, 128, 128]},
})

task = build_vqvae_task(cfg)
```

### MaskGITTask Constructor

**Before (deprecated):**

```python
task = MaskGITTask(
    model_config=maskgit_cfg,
    vqvae_ckpt_path="./checkpoints/vqvae.ckpt",
    hidden_size=768,
    num_layers=12,
    num_heads=12,
    mlp_ratio=4.0,
    dropout=0.1,
    gamma_type="cosine",
    lr=2e-4,
    weight_decay=0.05,
    warmup_steps=1000,
    sliding_window={"enabled": True},
)
```

**After (injected pattern):**

```python
from maskgit3d.runtime.composition import build_maskgit_task
from omegaconf import OmegaConf

cfg = OmegaConf.create({
    "task": {
        "_target_": "maskgit3d.tasks.maskgit_task.MaskGITTask",
        "vqvae_ckpt_path": "./checkpoints/vqvae.ckpt",
        "warmup_steps": 1000,
    },
    "model": {
        "_target_": "maskgit3d.models.maskgit.MaskGIT",
        "hidden_size": 768,
        "num_layers": 12,
        "num_heads": 12,
        "mlp_ratio": 4.0,
        "dropout": 0.1,
        "gamma_type": "cosine",
    },
    "optimizer": {"_target_": "torch.optim.AdamW", "lr": 2e-4, "weight_decay": 0.05},
})

task = build_maskgit_task(cfg)
```

## New Builder Pattern

The recommended way to construct tasks is via the builder functions in `maskgit3d.runtime.composition`:

| Function | Purpose |
|----------|---------|
| `build_vqvae_task(cfg)` | Build a fully assembled VQVAETask from Hydra config |
| `build_maskgit_task(cfg)` | Build a fully assembled MaskGITTask from Hydra config |
| `build_training_task(cfg)` | Generic task builder (legacy support) |
| `build_eval_task(cfg, ckpt_path)` | Build task from checkpoint for evaluation |

### What `build_vqvae_task` Assembles

1. **VQVAE model** via `create_vqvae_model(model_config)`
2. **VQPerceptualLoss** from task config (lambda_*, disc_*, perceptual_network)
3. **GANTrainingStrategy** from gradient_clip settings
4. **VQVAEReconstructor** with sliding window config
5. **VQVAETrainingSteps** composing all above
6. **GANOptimizerFactory** from optimizer config (if provided)

### What `build_maskgit_task` Assembles

1. **VQVAE** from checkpoint or default (frozen, eval mode)
2. **MaskGIT** transformer from model config
3. **MaskGITTrainingSteps** for training orchestration
4. **TransformerOptimizerFactory** (if optimizer config provided)

## Deprecation Timeline

| Version | Status |
|---------|--------|
| V1.x | Legacy constructor fully supported |
| V2.0 | Legacy constructor emits `DeprecationWarning`; still functional |
| V3.0 (planned) | Legacy constructor will be removed |

## Injection Detection

The injected component pattern is detected automatically:

```python
uses_injected_components = all(
    component is not None
    for component in (model, loss_fn, training_steps, optimizer_factory)
)
```

If all four components are provided, the task uses the injected path. Otherwise, it falls back to legacy construction with a deprecation warning.

## Attributes Set by Injection

### VQVAETask (injected path)

- `task.vqvae`
- `task.loss_fn`
- `task.training_steps`
- `task.gan_strategy`
- `task.reconstructor`
- `task.sliding_window_cfg`
- `task._downsampling_factor`
- `task.lr_g` / `task.lr_d`
- `task.gradient_clip_val` / `task.gradient_clip_enabled`

### MaskGITTask (injected path)

- `task.vqvae`
- `task.maskgit`
- `task.training_steps`
- `task.lr`
- `task.weight_decay`
- `task.warmup_steps`
- `task.optimizer_factory`

## Hydra Config Integration

The builder pattern integrates with Hydra config composition:

```yaml
# conf/task/vqvae.yaml
_target_: maskgit3d.tasks.vqvae_task.VQVAETask
use_perceptual: true
lambda_perceptual: 0.1
gradient_clip_val: 1.0
gradient_clip_enabled: true
```

```yaml
# conf/train.yaml
defaults:
  - task: vqvae
  - _self_

task:
  lr_g: 4.5e-06
  lr_d: 1e-04
```

The builder reads from `cfg.task`, `cfg.model`, `cfg.optimizer`, and `cfg.data` to assemble components.
